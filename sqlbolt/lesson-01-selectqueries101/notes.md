# LESSON 1 - SELECT QUERIES 101

## What I Learned
- SELECT statements, AKA *queries* allows us to retrieve data from a SQL database

## Important Syntax
- Select query for specific columns
`SELECT column, another_column, ...
FROM mytable;`

- Select query for **all** columns
`SELECT *
FROM mytable`
